# AI Campus Issue Reporting System

This project is a hackathon prototype that allows students to report campus issues through a simple web interface.

## Features
- Issue submission via web UI
- Real-time issue storage using Firebase
- AI-based issue categorization using Google Gemini (conceptual integration)

## Technologies Used
- HTML, JavaScript
- Google Firebase (Firestore)
- Google Gemini API

## Note
This is a prototype developed during the hackathon to demonstrate core functionality.
